(moving abstraction)

This is an open source application written for Fast Company Design. It is written in Processing.

This application was made to make videos to for Co.D header images. Instead of static images, this will make 1280 x 720 .ogg videos that will subtly play in the header image spot.

Flux Abstract beta 0.1 walkthrough from Scott Sullivan on Vimeo.

You need to install X11 if you want to use the app or run the sketches in Processing

OSX used to ship with X11, but it's not in Mountain Lion. If you're running Mountain Lion, you'll get this error message when you open the app.

Luckily, it's super easy- just hit continue here and it's just a few clicks after that.

Libraries you need to install in Processing (you don't need these if you just download the .app)

  • controlP5 by Andreas Schlegel
  • sDrop by Andreas Schlegel
  • GSVideo by Andres Colubri
